# IterativoVsRecursivo - Fatorial

Atividade acadêmica desenvolvida em Java com base no projeto `IterativoVsRecursivo`, utilizando a branch `faseh`.

## Objetivo

Implementar o cálculo do fatorial de um número utilizando duas abordagens:

- Método iterativo
- Método recursivo

## Arquivos alterados

- `src/Fatorial.java`
- `src/Main.java`

## Métodos implementados

### `calcularIterativo(int numero)`

Calcula o fatorial utilizando uma estrutura de repetição `for`.

### `calcularRecursivo(int numero)`

Calcula o fatorial utilizando recursividade, com caso base e chamada recursiva.

## Exemplo

Para o número `5`, o resultado esperado é:

5! = 120
